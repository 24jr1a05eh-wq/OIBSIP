# BMI Calculator

## Description
A Python application that calculates a person's Body Mass Index (BMI) based on height and weight.

## Features
- Calculate BMI
- Display BMI category
- Store records using SQLite
- Shows data in Graph Format

## Technologies Used
- Python
- SQLite

## Files
- bmi.py
- bmi_records.db
- requirements.txt

## How to Run

1. Install Python.
2. Open the project folder.
3. Run:

```bash
python .\bmi.py
```
