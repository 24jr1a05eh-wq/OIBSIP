# Password Generator

## Description
This is a Python-based Password Generator developed as part of the Oasis Infobyte Internship (OIBSIP). The application generates strong and secure random passwords based on the user's desired password length.

## Features
- Generate random passwords
- User-defined password length
- Uses uppercase and lowercase letters
- Includes numbers
- Includes special characters
- Easy-to-use interface

## Technologies Used
- Python 3
- Tkinter (if you created a GUI)
- Random module
- String module

## Installation

1. Clone the repository:

```bash

```

```bash

```

3. Install dependencies (if any):

```bash
pip install -r requirements.txt
```

```bash
python password_generator.py
```
