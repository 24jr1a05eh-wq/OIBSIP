# Weather App

## Description
The Weather App is a Python application that allows users to check the current weather conditions of any city. It retrieves weather information such as temperature, humidity, weather condition, and wind speed using a weather API.

## Features
- Search weather by city name
- Display current temperature
- Show humidity and wind speed
- Display weather description
- Simple and user-friendly interface

## Technologies Used
- Python
- Requests
- Tkinter (if you used a GUI)
