# BMI Calculator

## Description
A Python-based BMI Calculator that calculates a user's Body Mass Index (BMI) using height and weight. The application also stores BMI records in a SQLite database.

## Features
- Calculate BMI
- Display BMI category
- Store BMI records in SQLite database
- Simple and user-friendly interface

## Technologies Used
- Python
- SQLite

## Files
- bmi.py
- bmi_records.db
- requirements.txt
